# Figma

| 功能 | 快捷鍵（Mac） | 快捷鍵（Windows） |
| --- | --- | --- |
| 重播動畫 / 重新啟動 Prototype | `⌘` + `\` | `Ctrl` + `\` |
| 開啟/關閉 Prototype 預覽 | `⌘` + `\` | `Ctrl` + `\` |
| 顯示/隱藏設備框（手機框） | F | F |
| 快速切到 Design 面板 | Shift + D | Shift + D |
| 快速切到 Inspect 面板 | Shift + I | Shift + I |
| 開啟 Dev Mode | Shift + M | Shift + M |
| 連結元件 | Ctrl + Option + Command + K | Ctrl + Alt + K |
