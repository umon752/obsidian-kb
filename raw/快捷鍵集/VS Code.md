# VS Code

| 功能 | 快捷鍵（Mac） | 快捷鍵（Windows） |
| --- | --- | --- |
| 同時選取多個游標（多光標編輯） | `Option` + `滑鼠左鍵` | `Alt` + `滑鼠左鍵` |
| 選取下一個相同字串 | `⌘` + `D` | `Ctrl` + `D` |
| 選取所有相同字串 | `⌘` + `Shift` + `L` | `Ctrl` + `Shift` + `L` |
| 快速開啟檔案 | `⌘` + `P` | `Ctrl` + `P` |
| 快速搜尋整個專案（放大鏡） | `⌘` + `Shift` + `F` | `Ctrl` + `Shift` + `F` |
| 檔案內容搜尋 | `⌘` + `F` | `Ctrl` + `F` |
| 開啟命令面板（所有功能都在這，VS Code 入口） | `⌘` + `Shift` + `P` | `Ctrl` + `Shift` + `P` |
| 開啟/關閉 終端機 | `⌘` + ``` | `Ctrl` + ``` |
| 格式化文件 | `Option` + `Shift` + `F` | `Alt` + `Shift` + `F` |
| 顯示 / 隱藏側欄（Explorer） | `⌘` + `B` | `Ctrl` + `B` |
| Quick Fix | `⌘` + `.` | `Ctrl` + `.` |
| 單行刪除 | `⌘` + `Shift` + `K` | `Ctrl` + `Shift` + `K` |
| 向上/向下複製一行 | `Option` + `Shift` + `↑` / `↓` | `Alt` + `Shift` + `↑` / `↓` |
| 向上/向下移動一行 | `Option` + `↑` / `↓` | `Alt` + `↑` / `↓` |
| 摺疊程式碼區塊 | `⌘` + `Option` + `/ [`  | `Ctrl` + `Alt` + `/ [` |
| 展開程式碼區塊 | `⌘` + `Option` + `/ ]` | `Ctrl` + `Alt` + `/ ]` |
| 關閉目前開啟的檔案 | `⌘` + `W` | `Ctrl` + `W` |
| 完全退出 VS Code | `⌘` + `Q` | `Alt` + `F4` |
| 分割編輯區 | `⌘` + `\` | `Ctrl` + `\` |
| 分割特定數量編輯區 | `⌘` + `要分割幾個視窗數字` | `Ctrl` + `要分割幾個視窗數字` |
| 切換到下一個編輯區 | **`⌘` + `K` + `→`** | `Ctrl` **+** `K →` |
| 切換到上一個編輯區 | **`⌘` + `K` + `←`** | `Ctrl` **+** `K ←` |
| 切換到特定編輯區 | `⌘` + `第幾個編輯區數字` | `Ctrl` + `第幾個編輯區數字` |
| 切換同一編輯區內的下一個 Tab | `⌘` + `Option` + `→` | `Ctrl` + `Tab` |
| 切換同一編輯區內的上一個 Tab | `⌘` + `Option` + `←` | `Ctrl` + `Shift` + `Tab` |
| 多行游標 (垂直) | `⌘` + `Option` + `↑` / `↓` | **`Ctrl` + `Alt` + `↑` / `↓`** |
| 快速開啟檔案 | `⌘` + `P` | **`Ctrl` + `P`** |
| 移至行首/行尾 | `⌘` + `←` / `→` | `Home` / `End` |
| 移至文件最頂/最底 | `⌘` + `↑` / `↓` | `Ctrl` + `Home` / `End` |
| 跳過一個單字 | `Option` + `←` / `→` | `Ctrl` + `←` / `→` |
| 選取整行文字 | `⌘` + `Shift` + `←` / `→` | `Shift` + `Home` / `End` |
| 刪除游標位置到行首/行尾的所有文字 | `⌘` + `delete` / `delete x` | `Shift` + `Home` 後按 `Backspace` (或使用編輯器專屬組合鍵) |
