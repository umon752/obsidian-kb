# 自訂規範

## web components

<aside>
📌

nsdi-{元件名稱}

</aside>

> 元件名稱較長的話以「-」連接
> 

範例：

```html
<nsdi-field></nsdi-field>
<nsdi-accordion></nsdi-accordion>
<nsdi-stacking-toast></nsdi-stacking-toast>
```

## 自訂義事件命名方式

<aside>
📌

使用「:」、「-」組合

{元件名稱(名詞)}:{事件名稱(動詞)}

</aside>

> 動詞的部分如果需要放狀態再用「-」
> 

範例：

```jsx
new CustomEvent('nsdiAccordion:expand-before', {...})
new CustomEvent('nsdiAccordion:expand', {...})
```
