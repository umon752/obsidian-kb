為了在進後端之前先測試自己的樣板在收信軟體裡可以正確顯示，所以需要讓自己可以發信給自己，以下是一個簡單的做法（以 PHP 為例，倒不如說其他的我也不會，讚！）：

2. 發信的具體做法是透過 smtp 去把 mail 傳出去，所以會需要準備一個可以使用的 smtp，這裡選的是 google 的 smtp.gamil.com 來發信，每個帳號有每月 500 封的免費信可以發。
    
    ### **SMTP 的作用**
    
    SMTP 是發送電子郵件的核心協議，具體負責以下幾個工作：
    
    1. **郵件發送**：
        1. 用戶端（如你的郵件應用或伺服器程式）透過 SMTP 將郵件發送到郵件伺服器。
    2. **郵件轉發**：
        1. 如果收件人的郵件伺服器與發信人的伺服器不同，SMTP 會將郵件中繼到目標伺服器。
    3. **郵件接收（伺服器之間）**：
        1. 郵件伺服器之間使用 SMTP 進行溝通和轉送郵件。
    
    ---
    
    ### **SMTP 的運作流程**
    
    1. **客戶端連接 SMTP 伺服器**： 客戶端（如 Gmail、Outlook 或應用程式中的 PHPMailer）建立與 SMTP 伺服器的連線。
    2. **驗證身份**： 大多數 SMTP 伺服器需要提供用戶名和密碼來進行身份驗證。
    3. **郵件構建與發送**：
        1. 客戶端將郵件的相關資訊（發件人、收件人、主題、內容等）提交到 SMTP 伺服器。
        2. SMTP 伺服器將郵件傳送給目標伺服器。
    4. **收件伺服器接收**： 收件人的郵件伺服器接受郵件並存儲，以便用戶透過協議（如 IMAP 或 POP3）檢索郵件。
    
    ---
    
    ### **SMTP 的關鍵參數**
    
    在設定 SMTP 時，通常需要以下資訊：
    
    1. **伺服器地址（Host）**：
        1. SMTP 主機地址，例如 `smtp.gmail.com`、`smtp.example.com`。
    2. **埠號（Port）**：
        1. 常見埠號：
            - 25：傳統的 SMTP 埠（不加密，逐漸被淘汰）。
            - 465：SMTP over SSL（加密傳輸）。
            - 587：SMTP with STARTTLS（加密傳輸，推薦使用）。
    3. **加密方式**：
        1. **SSL（Secure Sockets Layer）**：在連線時啟動加密。
        2. **TLS（Transport Layer Security）**：先建立普通連線，再切換到加密。
    4. **身份驗證**：
        1. 使用者名稱和密碼，用於驗證 SMTP 發送的權限。
    
    ---
    
    ### **SMTP 常見用法**
    
    1. **電子郵件服務**： 用於大多數電子郵件應用程式（如 Gmail、Yahoo Mail）的郵件發送。
    2. **應用程式通知**：
        1. 網站或應用程式透過 SMTP 發送通知、確認郵件或重設密碼的郵件。
    3. **郵件行銷**：
        1. 發送大量電子郵件（例如 EDM，電子郵件行銷）時，SMTP 是核心技術。
    
    ---
    
    ### **SMTP 與其他郵件協議的區別**
    
    | **協議** | **用途** | **描述** |
    | --- | --- | --- |
    | SMTP | 發送郵件 | 用於電子郵件的發送和中繼傳輸。 |
    | IMAP | 接收郵件（伺服器同步） | 支援郵件伺服器同步，允許用戶從多個設備訪問同一郵件箱的內容。 |
    | POP3 | 接收郵件（下載） | 簡單協議，郵件從伺服器下載後即刪除伺服器副本，不支援多設備同步。 |
    
    ---

    ```
    <?php
    require 'src/plugins/phpmailer/src/PHPMailer.php';
    require 'src/plugins/phpmailer/src/SMTP.php';
    require 'src/plugins/phpmailer/src/Exception.php';
    
    use PHPMailer\PHPMailer\PHPMailer;
    use PHPMailer\PHPMailer\Exception;
    
    $mail = new PHPMailer(true);
    
    try {
        // 讀取EDM模板
        $edm_template = file_get_contents('template/edm-template.php');
        if($edm_template === false) {
            throw new Exception('無法讀取EDM模板');
        }
    
        // SMTP 設定
        $mail->isSMTP();                                // 使用 SMTP
        $mail->Host       = 'smtp.gmail.com';         // SMTP 主機 (替換為你的伺服器)
        $mail->SMTPAuth   = true;                       // 不啟用 SMTP 驗證
        $mail->Username   = 'yourname@nsdi.com.tw';   // SMTP 帳號
        $mail->Password   = '**** **** **** ****';           // SMTP 密碼（剛剛的應用程式密碼）
        $mail->SMTPSecure = PHPMailer::ENCRYPTION_SMTPS; // 加密方式：SSL
        $mail->Port       = 465;                        // SMTP 埠號：465 (SSL)
    
        // 發信人
        $mail->setFrom('yourname@nsdi.com.tw', '猜猜我是誰');
    
        // 收件人
        $mail->addAddress('bill@nsdi.com.tw', 'Bill'); // 收件人
    
        // 郵件內容
        $mail->isHTML(true);                                  // 使用 HTML 格式
        $mail->Subject = '測試郵件';
        $mail->Body    = $edm_template; // 郵件內容
        $mail->AltBody = '這是一封測試郵件，請勿回覆。';       // 純文字版本
    
        // 發送郵件
        $mail->send();
        echo '郵件發送成功';
    } catch (Exception $e) {
        echo "郵件發送失敗。錯誤訊息: {$mail->ErrorInfo}";
    }
    
    ?>
    
    ```
