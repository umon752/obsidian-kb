# 工具：Git

## 簡介
分散式版本控制工具，追蹤檔案變更、協作開發、管理分支。

## 設定檔
- 全域設定：`~/.gitconfig`

## 常用 alias
| alias | 原指令 |
|-------|--------|
| `st` | `status` |
| `ci` | `commit` |
| `br` | `branch` |
| `co` | `checkout` |
