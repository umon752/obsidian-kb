# Figma

## 簡介
Figma 是以瀏覽器為基礎的向量設計與原型製作工具，支援多人即時協作，為本團隊主要 UI/UX 設計工具。

## 特徵
- 支援 Prototype 預覽與動畫播放
- Dev Mode 提供前端查看設計稿的規格數值
- 可匯出 SVG icon 包給前端使用

## 使用規範
- 使用 px 為基本單位（非 pt）
- 電腦版設計稿寬度：1440px 或 1920px；手機版：375px
- Icon 建議統一 frame（如 24x24px）
