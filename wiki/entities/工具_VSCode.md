# 工具：VS Code

## 簡介
Microsoft 開發的跨平台輕量程式碼編輯器，透過 Extensions Marketplace 高度擴充，支援幾乎所有語言與工作流。

## 主要特色
- 強大的 Extensions 生態（語言支援、Git、AI、Sass、外觀等）
- 透過 `settings.json` 統一管理編輯器行為
- Snippets 系統支援自訂程式碼片段
- MCP 整合（Figma、Chrome DevTools、GitHub）
