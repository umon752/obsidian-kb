# 自訂規範

## 核心要點

### Web Components 命名
- 格式：`nsdi-{元件名稱}`（多字以 `-` 連接）
- 範例：`<nsdi-field>`、`<nsdi-accordion>`、`<nsdi-stacking-toast>`

### 自訂義事件命名
- 格式：`{元件名稱}:{事件名稱}`（名詞:動詞）
- 狀態以 `-` 附加：`nsdiAccordion:expand-before`、`nsdiAccordion:expand`
