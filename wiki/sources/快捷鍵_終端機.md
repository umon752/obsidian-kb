# 快捷鍵 — 終端機

## 核心要點

| 功能 | 快捷鍵 |
|------|--------|
| 跳到行首 | `Ctrl` + `A` |
| 跳到行尾 | `Ctrl` + `E` |
| 刪除游標左邊所有文字 | `Ctrl` + `U`（游標放行尾）|
| 刪除游標右邊所有文字 | `Ctrl` + `K`（游標放行首）|
| 中斷程序 | `Ctrl` + `C` |
| 強制退出（vim 等） | `Esc` → `:q!` → `Enter` |

### 指令補充
- 變更目錄內所有檔案權限：`chmod -R a+rw '目錄位置'`
