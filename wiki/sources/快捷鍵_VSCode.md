# 快捷鍵 — VS Code

## 核心要點

### 多游標 / 選取
| 功能 | Mac | Windows |
|------|-----|---------|
| 多游標 | `Option` + 點擊 | `Alt` + 點擊 |
| 選取下一個相同字串 | `⌘` + `D` | `Ctrl` + `D` |
| 選取所有相同字串 | `⌘` + `Shift` + `L` | `Ctrl` + `Shift` + `L` |
| 多行游標（垂直） | `⌘` + `Option` + `↑/↓` | `Ctrl` + `Alt` + `↑/↓` |

### 搜尋 / 導覽
| 功能 | Mac | Windows |
|------|-----|---------|
| 快速開啟檔案 | `⌘` + `P` | `Ctrl` + `P` |
| 搜尋整個專案 | `⌘` + `Shift` + `F` | `Ctrl` + `Shift` + `F` |
| 命令面板 | `⌘` + `Shift` + `P` | `Ctrl` + `Shift` + `P` |
| Quick Fix | `⌘` + `.` | `Ctrl` + `.` |

### 編輯
| 功能 | Mac | Windows |
|------|-----|---------|
| 格式化文件 | `Option` + `Shift` + `F` | `Alt` + `Shift` + `F` |
| 單行刪除 | `⌘` + `Shift` + `K` | `Ctrl` + `Shift` + `K` |
| 向上/下複製一行 | `Option` + `Shift` + `↑/↓` | `Alt` + `Shift` + `↑/↓` |
| 向上/下移動一行 | `Option` + `↑/↓` | `Alt` + `↑/↓` |
| 摺疊/展開區塊 | `⌘` + `Option` + `[/]` | `Ctrl` + `Alt` + `[/]` |

### 編輯區 / Tab
| 功能 | Mac | Windows |
|------|-----|---------|
| 分割編輯區 | `⌘` + `\` | `Ctrl` + `\` |
| 切換到下一個編輯區 | `⌘` + `K` + `→` | `Ctrl` + `K →` |
| 切換 Tab（下一個） | `⌘` + `Option` + `→` | `Ctrl` + `Tab` |
| 開關側欄 | `⌘` + `B` | `Ctrl` + `B` |
| 開關終端機 | `⌘` + `` ` `` | `Ctrl` + `` ` `` |
