# 快捷鍵 — Figma

## 核心要點

| 功能 | Mac | Windows |
|------|-----|---------|
| 重播動畫 / 開關 Prototype 預覽 | `⌘` + `\` | `Ctrl` + `\` |
| 顯示/隱藏設備框 | `F` | `F` |
| 切到 Design 面板 | `Shift` + `D` | `Shift` + `D` |
| 切到 Inspect 面板 | `Shift` + `I` | `Shift` + `I` |
| 開啟 Dev Mode | `Shift` + `M` | `Shift` + `M` |
| 連結元件 | `Ctrl` + `Option` + `⌘` + `K` | `Ctrl` + `Alt` + `K` |
