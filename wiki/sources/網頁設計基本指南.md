# 網頁設計基本指南

## 核心要點

### 基本規格
- 設計工具以 Figma 為主
- 單位：`px`（非 pt）
- 手機版寬度：375px；電腦版：1440px 或 1920px
- 字級避免小於 12px，建議預設 h1~h6、p、caption 字級變數
- 邊框避免非整數像素值（0.5px 等）

### 色彩變數命名
語意化命名，層級：`primary` > `secondary` > `accent` > `neutral` > 狀態色（`success/error/warning/info`）

### 圖片 / 影片 / Icon
- 圖片 < 1MB；影片比例 16:9
- Icon 建議統一 frame（如 24x24px），用 icomoon 產出 SVG icon 包

### 排版注意
- 有後台編輯的區塊，圖片不可壓到區塊範圍內
- 分頁筆數手機版與電腦版需一致

### 標準化模組（已整理待製作）
Input、Select2、Air-datepicker、麵包屑、Modal、Toast、Accordion
