# 其他雜項

## 核心要點

### YouTube iframe 參數
| 參數 | 說明 |
|------|------|
| `rel=0` | 影片結束後不顯示相關影片 |
| `autoplay=1` | 自動播放 |
| `mute=1` | 靜音播放 |
| `loop=1` | 重複播放 |
| `playlist=影片ID` | 重複播放必填 |
| `playsinline=1` | 取消 Safari 將影片放大 |
