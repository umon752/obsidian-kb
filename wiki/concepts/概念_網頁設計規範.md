# 網頁設計規範

## 定義
本規範整合設計師與前端工程師的協作標準，確保設計稿可順利轉換為可維護的程式碼。

## 核心規則

### 尺寸 / 版型
- 單位：`px`（非 pt）
- 手機版：375px；電腦版：1440px 或 1920px
- 字級 ≥ 12px，大標題不超過 100px
- 邊框粗細使用整數 px

### 色彩變數命名原則
語意化 > 功能性 > 具體描述
```
✅ primary-color、error-state、text-primary
❌ color1、color-blue、bg-color-1
```

### 設計 ↔ 前端協作
- 後台可編輯欄位需考量字數多少與 RWD 的排版
- 圖片不可壓入會伸縮的區塊範圍
- 分頁筆數手機版與電腦版需一致
- 動態需在切版前與前端確認（附參考網站或備註）
- 電子報內容寬度設計為 600px

### 需提供
- favicon、OG 圖
- 頁碼邏輯需標示於設計稿
