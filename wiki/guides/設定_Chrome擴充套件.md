# 設定：Chrome 擴充套件

## 安裝清單

### 追蹤 / 標籤類
| 擴充套件 | 用途 |
|---------|------|
| Tag Explorer | 快速瀏覽網站上所有第三方追蹤 / 標籤 |
| Tag Assistant | 檢查 Google 相關追蹤是否正確安裝 |
| LINE Tag Helper | 檢查 LINE 行銷追蹤是否正常 |
| Meta Pixel Helper | 檢查 Facebook / Instagram 廣告追蹤 |

### 偵錯類
| 擴充套件 | 用途 |
|---------|------|
| Redirect Path | 檢查網址是否有重新導向（Redirect） |

### 開發類
| 擴充套件 | 用途 |
|---------|------|
| Vue.js devtools | Vue 的開發者工具 |

## 注意事項
- ColorZilla（取色工具）已停用
