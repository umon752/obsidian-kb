# 設定：VS Code Extensions

## 輔助
| 名稱 | 功能 |
|------|------|
| Live Server | 啟動本機開發伺服器 |
| Path Intellisense | 自動補全檔案路徑 |
| Copy filename | 複製檔名 |
| Todo Tree | TODO 標記管理 |
| SVG Viewer | 預覽 SVG |
| Code Spell Checker | 程式碼拼字檢查 |
| Auto Rename Tag | 同步修改起始／結尾標籤 |
| Error Lens | 強化錯誤警告顯示 |
| Gremlins tracker | 顯示可能有害的隱藏字符 |
| Prettier | 程式碼格式化 |
| ESLint | JavaScript 語法檢查 |
| i18n Ally | 直接查看語系文字 |

## Sass
| 名稱 | 功能 |
|------|------|
| Sass (.sass only) | Sass / SCSS 語法高亮 |
| SCSS IntelliSense | 變數、Mixin、function 自動完成 |
| Live Sass Compiler | 即時編譯 SCSS → CSS |

## 外觀
| 名稱 | 功能 |
|------|------|
| Material Icon Theme | 檔案 icon 圖示 |
| indent-rainbow | 縮排顏色區別 |
| Rainbow Tags | 標籤巢狀層級上色 |
| Color Highlight | 顯示色碼顏色 |

## Git
| 名稱 | 功能 |
|------|------|
| Git Graph | Git 樹狀圖查看 |
| gitignore | 快速新增 .gitignore 條目 |

## AI

|------|------|
| GitHub Copilot | AI 程式碼補全 |
| GitHub Copilot Chat | AI 對話 |

## 語言

| UnoCSS | UnoCSS 自動完成 |
