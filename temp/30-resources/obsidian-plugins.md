# Obsidian 第三方插件清單

> 本文件列出此 Vault 目前啟用的所有 Community Plugins（第三方插件）。

---

## 插件列表

### 1. Git
| 欄位 | 內容 |
|------|------|
| Plugin ID | `obsidian-git` |
| 版本 | 2.38.0 |
| 作者 | [Vinzent](https://github.com/Vinzent03) |
| 說明 | 整合 Git 版本控制，支援自動備份與進階操作。 |

---

### 2. Custom Attachment Location
| 欄位 | 內容 |
|------|------|
| Plugin ID | `obsidian-custom-attachment-location` |
| 版本 | 10.3.4 |
| 作者 | [mnaoumov](https://github.com/mnaoumov/)（原作者：RainCat1998） |
| 說明 | 自訂附件儲存路徑，支援 `${noteFileName}`、`${date:format}` 等變數，類似 Typora 的行為。 |

---

### 3. Terminal
| 欄位 | 內容 |
|------|------|
| Plugin ID | `terminal` |
| 版本 | 3.23.0 |
| 作者 | [polyipseity](https://github.com/polyipseity) |
| 說明 | 在 Obsidian 內嵌入終端機（console / shell / terminal）。 |
